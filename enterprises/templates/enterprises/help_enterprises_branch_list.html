Enterprise Branches List Page - Help

 

This page displays a list of the selected enterprise or company branches that are captured in the MyWeXlog system.

The enterprise branches are listed in alphabetical order detailing the Branch Name, the branch type and the Current and Past Employees as recorded in MyWeXlog.

The 'Rating' for the enterprise branch is reflective of the rating allocated by the talent as indicated by the 'Number of vates' that has done work for the branch. If you click on the icon it will open the rating summary with the breakdown per the following:

  • Employer Performance
  • Payment Receipt
  • Repeat Work Potential
  • Payment Duration
  • Average

These ratings are calculated from an average of a minimum of five ratings.

You have the option to:

  • Add Branch - Button will require you to capture the Company that the branch belongs to, the Branch or Division Name, the Branch Type, Physical Address and Industry.
  • Detail - Button will open the branch detail page with the Branch Name, the Branch Type, Physical Address and Industry.
  • Back - Navigate back to the Enterprises Home Page.
