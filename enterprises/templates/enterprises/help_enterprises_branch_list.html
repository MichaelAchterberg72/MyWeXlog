Enterprise Branches List Page - Help

 

This page displayes a list of the selected enterprise or company branches that are captured in the WeXlog system.

The enterprise branches are listed in alphabetical order detailing the Branch Name, the branch type and the Current and Past Employees as recorded in WeXlog.

You have the option to:

  • Add Branch - Button will require you to capture the Company that the branch belongs to, the Branch or Division Name, the Branch Type, Physical Address and Industry.
  • Detail - Button will open the branch detail page with the Branch Name, the Branch Type, Physical Address and Industry.
  • Back - Navigate back to the Enterprises Home Page.
