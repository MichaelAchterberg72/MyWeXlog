Add a New Enterprise Details Page - Help

 

This page displays the fields that are required to capture a new Enterprise.

The fields required are:

Enterprise Name Enter the full name of the main company, if you are working for a branch the details of branch offices will be captured at a later point.
Website Enter the company's website address.
Enterprise Description Enter a description of the main company.

**In the event that one of the dropdown lists is not populated please refresh the page.

A * next to the field name indicates that the field is a required field.
