Project Detail Page - Help

 

This page displayes a the details of the selected project that you clicked the Detail button.

The view displays the Project Owner, the Project Name, Industry, the total number of hours recorded in WeXlog for the hours worked on the project as well as the project country, city and region. The view also list the number of WeXlog members that have worked on the project and recorded their experience for the project.

The Edit Project button will open the project details page allowing you to amend the recorded information of the project.

You have the option to:

  • Wexlog Hours - View a listing of the experience items that have been captured that make up the hours spent on the project.
  • Back - allows you to go back to the Projects Home page.
