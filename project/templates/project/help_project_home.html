Project Home Page - Help

 

This page displayes a list of the projects that are captured in the WeXlog system.

The projects are listed in alphabetical order detailing the Project Name, Project Industry and the Project Country.

The See Detail button will open the project details page which will detail the Project Owner, Project Name, Industry, Total Project Hours, Country, City, Region and the Total number of people on the project as listed in WeXlog.

You have the option to:

  • Add Project - Add a new project which will request you to supply the Project Name, Project Owner, Project Site, Industry, Country, Region and a City.
  • Search Projects - Using a Keyword search the projects based on the Project Name, Company Name, Industry, Country, Region and City.
  • Back - allows you to go back to the Enterprises Home page.
