Add a New Book Read Page - Help

 

This page displayes the fields that are required to capture a new book read.

The fields required are:

Book This will be selected from the dropdown list and if you add a new book it will automatically populate the field.
The Add a new Book link will open a popup window and request you to add the book title, type, publisher, author, genre, url and skills.
Book Format This will be selected from the dropdown list and should be a hard copy, pdf, kindle, etc.
The Add a new book format link will request you to add the format type if your desired option is not in the list.
Date Finished Here please enter the date that you finished reading the book. Open the date picker by clicking on the up or down arrows that appear on the right of the capture field when you hover over it.

**In the event that one of the dropdown lists is not populated please refreash the page.

Click the button when complete.

You have the option to:

  • Back - allows you to navigate back to the Booklist Home page.
