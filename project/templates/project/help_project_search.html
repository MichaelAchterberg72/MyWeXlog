Project Search Page - Help

 

This page displays a list of the projects that are captured in the MyWeXlog system that meet your keyword parameter.

The search function will search against the Project Name, the Company Name, the Industry, the Country, Region and City.

The projects are listed in alphabetical order of the Project Owner Company Name detailing the Project Name, Project Industry and the Project Country.

The Project Detail button will open the Details listing for the project.

You have the option to:

  • Add Project - Add a new project which will request you to supply the Project Name, Project Owner, Project Site, Industry, Country, Region and a City.
  • Search Projects - Using a Keyword search the projects based on the Project Name, Company Name, Industry, Country, Region and City.
  • Back - allows you to go back to the Projects Home page.

The Search Again button will refresh the search page to the initial search form requesting you to enter your search keyword.
